 ཀ དཔལ་ཡེ་ཤེས་ཀྱི་མགོན་པོ་བརྒྱད་ཀྱི་ཞལ་ཉེ་བར་མཐོང་ཚུལ་གྱི་ལོ་རྒྱུས་ངེས་པ་རྣམ་རོལ། ཁ རྡོ་རྗེ་ནག་པོ་ཆེན་པོ་བརྒྱད་ཀྱི་མན་ངག་ཐམས་ཅད་ཀྱི་རྒྱུད། ག མགོན་པོ་མཆེད་བརྒྱད་ཀྱི་དཀྱིལ་འཁོར་གྱི་ཐིག་རྩ། ང ཐུབ་པའི་བསྟན་པ་སྲུང་བའི་ཡེ་ཤེས་ཀྱི་མགོན་པོ་མཆེད་བརྒྱད་ཀྱི་ཞལ་ཉེ་བར་བལྟ་བའི་དབང་གི་ཆུ་བོ་ཆེན་པོ། ཅ དཔལ་ཡེ་ཤེས་ཀྱི་མགོན་པོ་བརྒྱད་ཀྱི་དབང་བཞི་བྱིན་རླབས་ཀྱི་ཚུལ་དུ་བསྐུར་བ་དངོས་གྲུབ་ཀྱི་དགའ་སྟོན། ཆ དཔལ་ཡེ་ཤེས་ཀྱི་མགོན་པོ་ཕྱག་བཞི་པའི་རྗེས་གནང་ཕྲིན་ལས་ལྷུན་གྲུབ། ཇ དཔལ་ཡེ་ཤེས་ཀྱི་མགོན་པོ་བརྒྱད་ཀྱི་ཕྲིན་ལས་དངོས་གྲུབ་འདོད་འཇོ། ཉ ཐུབ་པའི་བསྟན་པ་སྲུང་བའི་དཔལ་ཡེ་ཤེས་ཀྱི་མགོན་པོ་བརྒྱད་ཀྱི་ཞལ་ཉེ་བར་བལྟ་བའི་མན་ངག་ལོག་འདྲེན་ཚར་གཅོད་དངོས་གྲུབ་སྩོལ་བའི་ནོར་བུ་དགོས་འདོད་ཀུན་འབྱུང༌། ཏ ཡེ་ཤེས་མགོན་པོ་མཆེད་བརྒྱད་ཀྱི་ཞལ་ཉེ་བར་བལྟ་བའི་མན་ངག་གི་བསྐང་གསོ་དངོས་གྲུབ་འབྱུང་བའི་བང་མཛོད། ཐ དཔལ་རྡོ་རྗེ་ནག་པོ་ཆེན་པོ་བརྒྱད་ཀྱི་དངོས་གྲུབ་སྲུང་ཟློག་གི་མན་ངག་དགྲ་བགེགས་ངེས་པར་འཇོམས་པར་བྱེད་པ་གནམ་ལྕགས་འབར་བའི་མཚོན་ཆ། ད མགོན་པོ་བརྒྱད་ཀྱི་མན་ངག་སྲོག་གི་སྤུ་གྲིའི་ཡིག་ཆུང་སྔགས་དང་དམིགས་པའི་ཁ་སྒྱུར། ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ 233 ན ཐུབ་པའི་བསྟན་སྲུང་མགོན་པོ་བརྒྱད་ཀྱི་སྒོ་ནས་དགྲ་བགེགས་སྒྲོལ་བའི་མན་ངག་སྲོག་གི་སྤུ་གྲི། ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ 239 པ ནག་པོ་ཆེན་པོ་མཆེད་བརྒྱད་ཀྱི་ལས་མཐའ་ཉེར་མཁོར་བསྡུས་པའི་ཕན་ཐབས་མན་ངག་མཆོག་དམ་པ་སྐྱོན་གྱི་གྲངས་ཅན། ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ 245 ཕ ཐུབ་པའི་བསྟན་སྲུང་དཔལ་ཡེ་ཤེས་ཀྱི་མགོན་པོ་བརྒྱད་ཀྱི་ཞལ་ཉེ་བར་བལྟ་བའི་ལས་མཐའ་མན་ངག་གཏེར་གྱི་བང་མཛོད། ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ 257 བ དཔལ་ཡེ་ཤེས་ཀྱི་མགོན་པོ་བྱ་རོག་གི་མཚན་ཅན་གྱི་གཏོར་ཆེན་བརྒྱ་རྩའི་ལག་ལེན་གསལ་བ་ལས་རིམ་གསལ་སྒྲོན་ལ་སྦྱར།

མ མགོན་པོའི་ལས་རིམ་གསལ་སྒྲོན་གྱི་མཆོད་པ་ཁ་འགེངས་ཡིག་རྙིང་ལྟར། ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ 285 ཙ བདེ་མགོན་གྱི་ལས་རིམ་གསལ་བྱེད་སྒྲོན་མེའི་ལྷན་ཐབས་བསྙེན་སྒྲུབ་ཀྱི་མན་ངག ། ཚ གཏོར་ཟློག་བྱ་མགོ་མའི་ལས་བསྒྲིགས་བསྟན་སྲུང་མཚོན་ཆའི་ཉེར་མཁོ་ལྷན་ཐབས། ཛ མ་མགོན་གཉིས་ཀྱི་གཏོར་བསྔོས་བསྡུས་པ། ཝ བསྐང་བའི་ཆོ་ག་ཀུན་ཏུ་བཟང་པོའི་རོལ་མཚོ། ཞ སྟོབས་འཕྲོག་ནག་པོའི་བསྙེན་སྒྲུབ་བསྡུས་པ། ཟ ལྷ་ཆེན་སྟོབས་འཕྲོག་ནག་པོའི་མངའ་གསོལ། འ གཉན་མགོན་ཞལ་བཞི་པའི་གཏོར་བསྔོས་དང་བསྙེན་སྒྲུབ། ཡ དཔལ་ཡེ་ཤེས་ཀྱི་མགོན་པོ་བེང་ཆེན་ལྕམ་དྲལ་གྱི་དབང་དང་ལས་བདག་ཕོ་མོའི་བཀའ་གཏད་དང་བཅས་པ་བྱིན་རླབས་གཏེར་མཛོད། ར དཔལ་ཡེ་ཤེས་ཀྱི་མགོན་པོ་བེང་ཆེན་ལྕམ་དྲལ་གྱི་ཕྲིན་ལས་ཀྱི་རིམ་པ་དངོས་གྲུབ་ཀུན་འབྱུང། ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ ་ 387 ལ དཔལ་ཡེ་ཤེས་ཀྱི་མགོན་པོ་བེང་ཆེན་ལྕམ་དྲལ་ལས་བདག་ཕོ་མོ་དང་བཅས་པའི་གཏོར་མས་ལས་རབ་འབྱམས་སྒྲུབ་པ་བར་གཅོད་ཀུན་སེལ། ཤ དཔལ་ཡེ་ཤེས་ཀྱི་མགོན་པོ་བེང་ཆེན་ལྕམ་དྲལ་ལས་བདག་ཕོ་མོ་དང་བཅས་པའི་པར་བུ་ནག་པོའི་སྒྲུབ་ཐབས་གནམ་ལྕགས་ཐོག་མདའ། ས དཔལ་ཡེ་ཤེས་ཀྱི་མགོན་པོ་བརྒྱད་ཀྱི་རྫོགས་རིམ་རྡོ་རྗེའི་ཚིག་རྐང།
